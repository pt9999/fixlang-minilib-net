### namespace Minilib.Net.HTML::HTMLHelpers

#### escape_html

Type: `Std::String -> Std::String`

Escapes HTML special characters.
eg. `&` -> `&amp;`, `<` -> `&lt;`, `>` -> `&gt;`, `\"` -> `&quot;`, `'` -> `&#039;`

#### unescape_html

Type: `Std::String -> Std::String`

Unescapes HTML special characters.
eg. `&amp;` -> `&`, `&lt;` -> `<`, `&gt;` -> `>`, `&quot;` -> `\"`, `&#039;` -> `'`.
NOTE: Other character references is also converted.
